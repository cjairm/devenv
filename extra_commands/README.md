# Useful commands

## Commands browser

```
Command-Shift + [ -> Move to tab backward
Command-Shift + ] -> Move to tab forward
Command + t -> Create tab
Command + w -> Close tab
```

## Keyboard - Terminal

```
Ctrl-Raise + s -> Move word by word backward
Ctrl-Raise + f -> Move word by word forward
```
